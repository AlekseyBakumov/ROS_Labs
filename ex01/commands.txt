=== 1
Название: find
Описание: Рекурсивный поиск файлов по заданному фильтру
Синтаксис: find [опции] начальная точка фильтр
Пример: find ./ex00 -name "*.md"
Примечание: Умеет работать с регулярными выражениями (^ $ . [ ] * \ -)
=== 2
Название: grep
Описание: Интерактивный монитор процессов для Unix-систем
Синтаксис: grep [опции] шаблон файл
Пример: grep "terminal command" ./ex00/READDME.md
Примечание: Часто используется в сочетании с другими командами. Прим.: ls /bin | grep -w zip
=== 3
Название: chmod
Описание: Изменяет права доступа к файлам или каталогам
Синтаксис: chmod [опции] режим файл
Пример: chmod 777 README.md
Примечание: Режим состоит из трех цифр: rwx (Права на чтение, запись и исполнение)
=== 4
Название: tar
Описание: Создание и распаковка архивов
Синтаксис: tar [опции] файл_архива [файлы]
Пример: tar cf ex00.tar /ex00
Примечание: -c создать, -f файл
=== 5
Название: scp
Описание: Безопасное копирование файлов между устройствами по ssh
Синтаксис: scp [опции] источник назначение
Пример: scp ./ex00/screenshot.png user@remote-host:/tmp/screenshot.png
Примечание: Похож на cp, но для удаленных машин
=== 6
Название: less
Описание: Постраничный просмотр текста в файле с возможностью прокрутки
Синтаксис: less файл
Пример: less ./ex00/README.md
Примечание: Для выхода нажать q
=== 7
Название: head
Описание: Вывод первых строк файла
Синтаксис: head [опции] файл
Пример: head -n 2 ./ex00/README.md
Примечание: -n для указания конкретного числа строк
=== 8
Название: tail
Описание: Вывод последних строк файла
Синтаксис: tail [опции] файл
Пример: tail -n 2 ./ex00/README.md
Примечание: -n для указания конкретного числа строк
=== 9
Название: mount
Описание: Монтирование файловой системы
Синтаксис: mount -t тип девайс каталог
Пример: mount /dev/sdb1 /media/usb_drive
Примечание: Используется, например, при установке нового диска
=== 10
Название: kill
Описание: Позволяет "убить" процесс по его id (pid)
Синтаксис: kill [опции] идентификатор
Пример: kill 256
Примечание: На самом деле, он просто отправляет сигнал TERM в процесс.
